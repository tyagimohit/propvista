# RAJ HOMES

A premium, responsive single-page real-estate website for Raj Nagar Extension, Ghaziabad. It uses only HTML, CSS and vanilla JavaScript, so it can be published directly on GitHub Pages.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository.
2. In **Settings → Pages**, select **Deploy from a branch**.
3. Select the main branch and `/ (root)` folder.
4. Set the canonical URL in `index.html`, `robots.txt`, and `sitemap.xml` to the final domain.

## Notes

- Images are served from Unsplash URLs to keep the repository lightweight. Download and replace them in `assets/images/` if a fully offline deployment is preferred.
- The contact form demonstrates client-side validation and confirmation only. Connect it to your preferred form endpoint or CRM before production.
- Review project names, pricing, RERA details, contact information, and location timings before launch.
